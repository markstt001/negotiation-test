### 2.2 致命弱点情感化

**当前：**
```
⚠️ 过度分析导致错失最佳采购时机
📍 易暴露：紧急采购、市场快速变化时
💡 改进：设定决策截止时间，超过时限就行动
📝 示例：告诉自己"今天下午 3 点前必须决定"
💎 另一面：谨慎是你的超能力，只是需要平衡
```

**优化后：**
```
⚠️ 你有时候，会被自己的谨慎困住

是不是经常这样：
明明已经分析了 3 天，数据也够了，
但总觉得"还不够充分"。
结果眼睁睁看着机会溜走。

这不是你的错。
你的谨慎，保护了公司很多次。
只是有时候，完美是完成的敌人。

试试这样：
下次遇到紧急采购，告诉自己：
"今天下午 3 点前必须决定。"
时间一到，就拍板。

你会发现：
80 分的决定 + 快速行动，
比 100 分的决定 + 错过时机，
要好得多。

你的谨慎是超能力，
只是需要学会什么时候放下它。
```

**文件：** `weaknesses-emotional.js`
