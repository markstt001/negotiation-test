## 使用流程

### 用户测试后分享流程：
1. 完成测试，查看结果页面
2. 点击"📸 生成海报分享"按钮
3. 海报在弹窗中显示，可预览效果
4. 点击"💾 长按保存图片" → 保存到相册
5. 打开微信 → 粘贴分享文案 → 发送图片到朋友圈

### 微信分享配置（可选）：
如需微信分享时显示自定义标题和封面图，需要：
1. 在微信公众号后台配置 JS-SDK 安全域名
2. 后端实现微信签名接口
3. 前端调用 `wx.config()` 和 `wx.updateAppMessageShareData()`

当前版本已包含完整的分享文案生成和复制功能。
